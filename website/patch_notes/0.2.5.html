V0.2 Finalizing

V0.2 Finalizing is a relatively small update adding features and fixing some bugs.
This was to set the stage for an update much, much larger.

New Content

Keybinds

‣ Added 3 keybinds. These 3 keybinds are U, ~, and B. The U keybind would quickly shorten the song URL, the ~ keybind (or shift `) creates a new blank song (the keybind is unfinished and thus doesn't actually make a new blank song), and the B keybind was yoinked from Ultrabox, sending you to the 'beats per bar' prompt.

Presets

‣ Added 2 presets. They were added as the following:
|
[Midbox Misc Presets (2)] - Mechanical Blip and Another Piano.

Misc

‣ Made a forms for users to report bugs and send suggestions/feedback for Midbox and linked it in the description.

‣ Added a Midbox theme, which was unfinished at the time.

‣ Added a language preference that would change the language between English and Spanish. Most translations were unfinished.
In the current version of Midbox, the Spanish translations have been finished.

Bug Fixing

Misc

‣ Fixed an issue where FM operator frequencies would be defaulted on 0.125x rather than 1x.

‣ Fixed an issue where chip waves would be offset on various presets.

Changes

Misc

‣ Changed the color of the mod name.
