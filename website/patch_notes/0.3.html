Midbox Revolution

Midbox Revolution, or Midbox V0.3, is the largest update to come to Midbox yet.
This update takes various things from other Beepmods, ports over Jummbox V2.6's content/bugfixes, and adds many unique things.

New Instrument Types

‣ The Supersaw from Beepbox V4.2!
An instrument type that uses many sawtooth/pulse width waves that shift in frequency using adjustable chorus-like settings.
Comes with modulators for each of its settings.
Fun fact: Midbox was the first mod other than Jummbox to port the supersaw to Jummbox.

‣ The Advanced FM from Goldbox!
This instrument is a more complex counterpart to the regular FM, using 6 operators instead of 4, and allowing algorithms/feedbacks to be fully customizable.
Comes with a few extra modulators and envelopes for it's new sliders.

‣ The Wavetable!
An instrument type that one-ups Sandbox's duty cycle instrument type. It uses 32 custom chips in a cycle where they swap through each other with adjustable speed.
Comes with 2 modulators to control the speed and set the current wave of the cycle.
The wavetable also has an interpolation checkbox underneath its speed slider that allows the waves to smoothly transition between eachother, as well as two more checkboxes for if the wavetable should reset the cycle per note, and if so, the other for making the cycle end at the final wave.

New Effects

‣ The Percussion effect!
An effect for preventing percussive instruments from changing pitch to song-wide pitch changes. These are: Song Key, Song Key Octave, and Song Detune Modulation.

‣ The Wavefold effect!
An effect for making an instrument's waveform invert direction when hitting certain boundaries.

Effects | Chord Type

‣ Added the strum speed slider, as well as its modulator.

‣ Added 'arpeggio patterns' for arpeggio chords.
It is a built-in pattern system which the arpeggio will use to cycle through notes with in an arpeggio chord.

Effects | Transition Type

‣ Added the slide speed slider, as well as its modulator.

‣ Added a checkbox that toggles whether a seamless transition is cut off at the end of a pattern.

Keybinds

‣ Added 5 keybinds.
These keybinds are B, X, `, CTRL+CLICK, and CTRL+R.
The X keybind 'cuts' the pattern, removing all notes from it while also copying them. The B keybind now activates a temporary loop bar mode, while the 'beats per bar' prompt keybind was moved to 'SHIFT+B'. The ` keybind opens the 'song recovery' prompt, holding CTRL and drag clicking on an filter/slider will record the value to a modulator as the song plays, and CTRL+R takes you to the new random generation settings prompt.

‣ Finished the ~ (SHIFT+`) keybind for starting a new blank song.

‣ Added a prompt in the preferences menu that allows toggling keybinds on/off called 'Keybind Setup...'.

Unisons

‣ Added unison compatibility to the pulse width, spectrum, and noise instrument/drum types.

‣ Added 2 unisons.
They are Bent and Hecking Gosh (Goldbox).

Chipwaves

‣ Added 6 chipwaves.
These chipwaves are Accurate Sine, Accurate Triangle, Accurate Trapezoid, Secant, Trapezoid (Goldbox), and Glitch 2.

Envelopes

‣ Tweaked envelope numbers and speed values a bit as well as adding many more numbers.

‣ Added 11 envelopes.
The new envelopes are: Linear, Rise, Wibble, Mini Tremolo, Full Tripolo, Semi Tripolo, Mini Tripolo, Decelerate, Stairs, Looped Stairs, and Jummbox Blip.
The Linear, Rise, and Wibble envelopes were ported from Goldbox. Jummbox Blip was ported from Jummbox V2.6. Decelerate, Stairs, and Looped Stairs are unique envelopes to Midbox.

‣ Added a dropdown containing more envelope settings. (Jummbox Addition)
These settings are envelope speed (affects all envelopes on the instrument) and discrete envelopes (makes envelopes sound harsher).

‣ Renamed the 'Slow Swell' envelope to 'Swell 5'.

FM-Centric

‣ Added 3 FM operator frequencies.
These are 48×, 64×, and 128×.

‣ Added 6 FM operator waveforms.
These are Clang, Metal, White Noise, Secant, Double Sine, and Rounded. The rounded waveform was ported from Ultrabox.

‣ Remade the FM pulse width to act like a normal pulse width, as well as giving it modulation and envelope support. (Ultrabox Addition)

‣ Added many more algorithm and feedback options to the FM.

Custom Chip-Centric

‣ Added copy/paste wave buttons in the custom chip's UI and prompt.

‣ Added drawing tools to the custom chip's prompt.
There are 4 tools/modes; Cursor, Line, Curve, and Selection.
- Cursor is the default draw mode where you click to place a box representing one of the custom chip's samples.
- The line tool draws a straight line from where you first clicked to where you release it.
- The curve tool first draws a line on the first drag click, and then on the second drag click, bends that line based on the mouse position.
- The selection tool creates a box, which you can drag and use keybinds in to change a specified part of the custom chip.
‣ Added the load preset button into the custom chip prompt.

‣ Added a randomization button in the custom chip prompt which randomizes the waveform.

‣ Various keybinds now work in the custom chip prompt. E.g: + and - shift wave samples up/down. R randomizes waveform. C and V copy/paste the waveform.

Basic Noise-Centric

‣ Made basic noise both an instrument type and drum type.

‣ Ported 7 noise types from Goldbox, Old Modbox, and Sandbox.
The new noise types are: Static, Retro Clang, Harsh, Trill, Detuned Periodic, Snare, and Chime.

Randomly Generated Instruments

‣ Added all features to the random generator.

‣ Added custom chips and drumsets into the list of possible randomly generated instruments.

‣ Added a prompt for better control on how your instrument randomly generates.
It can be found in the edit menu or by using the CTRL+R keybind.

‣ Added speed displays next to all speed sliders. (Jummbox Addition)
They can be measured in ticks, multipliers, or waves per beat (wavetable-only case).
The affected sliders are: Arpeggio Speed (multiplier), Vibrato Speed (multiplier), Strum Speed (ticks), Slide Speed (ticks), Envelope Speed (multiplier), and Wavetable Speed (WPB).

Removed

‣ Strum-Based Chord Types

This refers to the various strum-like chords that varied in speeds.
They were removed due to not being needed anymore upon the addition of strum speed.
[REMOVED]: Medium Strum, Slow Strum, ⅓ Strum, ⅓ Medium Strum, ⅓ Slow Strum, and Build Up chord types.

‣ 'In Pattern' Transition Types

This refers to the transition types that would be cut off at the end of a pattern.
They were removed due to not being needed anymore upon the addition of the 'continue through pattern/in pattern' checkbox.
[REMOVED]: Continue in Pattern and Slide in Pattern transition types.

‣ Insta-Slide Transition Type

This refers to the transition type that was a very fast version of slide.
It was removed due to not being needed anymore upon the addition of slide speed.
[REMOVED]: Insta-Slide transition type.
